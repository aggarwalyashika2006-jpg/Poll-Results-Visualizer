## 📌 Overview
The Poll Results Visualizer is an interactive data analytics project that converts raw survey/poll data into meaningful insights using Python.

It simulates real-world survey systems and demonstrates how organizations analyze user preferences, trends, and patterns for decision-making.

---

## 🎯 Problem Statement
Raw survey data is:
- Difficult to interpret  
- Time-consuming to analyze manually  
- Lacks clear actionable insights  

---

## 💡 Solution
This project:
- Cleans and processes poll data  
- Performs statistical analysis (vote counts, percentages)  
- Visualizes insights through charts  
- Provides an interactive dashboard  

---

## 🧠 Key Features
- 📊 Bar Chart (vote distribution)
- 🥧 Pie Chart (vote percentage)
- 🌍 Region-wise analysis
- 👥 Demographic filtering (Age, Gender)
- 📈 Trend analysis over time
- 🔍 Interactive filters
- 📌 KPI metrics (Top choice, total votes)
- ⬇️ Download filtered data
- 🌐 Live deployed dashboard

## ▶️ How to Run Locally

### 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/Poll-Results-Visualizer.git

cd Poll-Results-Visualizer
### 2. Install dependencies

pip install -r requirements.txt


### 3. Generate dataset

python main.py


### 4. Run the dashboard

python -m streamlit run dashboard/app.py
